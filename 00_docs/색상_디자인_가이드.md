# WEAV AI 색상 디자인 가이드

다른 서비스에서 WEAV AI와 동일한 색상 디자인을 구성하고 싶을 때 사용할 수 있는 명확한 가이드입니다.

---

## 1. 색상 시스템 개요

WEAV AI는 **HSL 색상 변수 기반의 다크 테마 디자인 시스템**을 사용합니다.

- **색상 형식**: HSL (Hue, Saturation, Lightness)
- **테마**: 다크 모드 (Dark Mode)
- **스타일링 프레임워크**: Tailwind CSS (CSS 변수 사용)

---

## 2. 정확한 요청 방법

### 방법 1: HSL 값 직접 제공 (가장 정확)

다음과 같이 요청하세요:

> "다음 HSL 색상 변수를 사용하여 WEAV AI와 동일한 다크 테마 색상 시스템을 구성해주세요:
> 
> **배경 및 기본 색상:**
> - 배경: `hsl(222.2, 84%, 4.9%)` (매우 어두운 남청색)
> - 전경(텍스트): `hsl(210, 40%, 98%)` (거의 흰색)
> 
> **카드 및 팝오버:**
> - 카드 배경: `hsl(222.2, 84%, 4.9%)`
> - 카드 텍스트: `hsl(210, 40%, 98%)`
> - 팝오버 배경: `hsl(222.2, 84%, 4.9%)`
> - 팝오버 텍스트: `hsl(210, 40%, 98%)`
> 
> **주요 색상:**
> - Primary: `hsl(210, 40%, 98%)` (배경: `hsl(222.2, 47.4%, 11.2%)`)
> - Secondary: `hsl(217.2, 32.6%, 17.5%)` (텍스트: `hsl(210, 40%, 98%)`)
> - Accent: `hsl(217.2, 32.6%, 17.5%)` (텍스트: `hsl(210, 40%, 98%)`)
> - Muted: `hsl(217.2, 32.6%, 17.5%)` (텍스트: `hsl(215, 20.2%, 65.1%)`)
> 
> **경고 및 테두리:**
> - Destructive (경고/삭제): `hsl(0, 62.8%, 30.6%)` (텍스트: `hsl(210, 40%, 98%)`)
> - 테두리: `hsl(217.2, 32.6%, 17.5%)`
> - 입력 필드: `hsl(217.2, 32.6%, 17.5%)`
> - 포커스 링: `hsl(212.7, 26.8%, 83.9%)`"

### 방법 2: CSS 변수 파일 제공

다음 CSS 코드를 제공하고 "이 CSS 변수를 사용하여 동일한 색상 시스템을 구성해주세요"라고 요청하세요:

```css
:root {
  --background: 222.2 84% 4.9%;
  --foreground: 210 40% 98%;

  --card: 222.2 84% 4.9%;
  --card-foreground: 210 40% 98%;

  --popover: 222.2 84% 4.9%;
  --popover-foreground: 210 40% 98%;

  --primary: 210 40% 98%;
  --primary-foreground: 222.2 47.4% 11.2%;

  --secondary: 217.2 32.6% 17.5%;
  --secondary-foreground: 210 40% 98%;

  --muted: 217.2 32.6% 17.5%;
  --muted-foreground: 215 20.2% 65.1%;

  --accent: 217.2 32.6% 17.5%;
  --accent-foreground: 210 40% 98%;

  --destructive: 0 62.8% 30.6%;
  --destructive-foreground: 210 40% 98%;

  --border: 217.2 32.6% 17.5%;
  --input: 217.2 32.6% 17.5%;
  --ring: 212.7 26.8% 83.9%;
}
```

### 방법 3: Tailwind 설정 파일 제공 (Tailwind 사용 시)

Tailwind를 사용하는 경우, 다음 설정 파일을 제공하세요:

```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        border: 'hsl(var(--border))',
        input: 'hsl(var(--input))',
        ring: 'hsl(var(--ring))',
        background: 'hsl(var(--background))',
        foreground: 'hsl(var(--foreground))',
        primary: {
          DEFAULT: 'hsl(var(--primary))',
          foreground: 'hsl(var(--primary-foreground))',
        },
        secondary: {
          DEFAULT: 'hsl(var(--secondary))',
          foreground: 'hsl(var(--secondary-foreground))',
        },
        destructive: {
          DEFAULT: 'hsl(var(--destructive))',
          foreground: 'hsl(var(--destructive-foreground))',
        },
        muted: {
          DEFAULT: 'hsl(var(--muted))',
          foreground: 'hsl(var(--muted-foreground))',
        },
        accent: {
          DEFAULT: 'hsl(var(--accent))',
          foreground: 'hsl(var(--accent-foreground))',
        },
        popover: {
          DEFAULT: 'hsl(var(--popover))',
          foreground: 'hsl(var(--popover-foreground))',
        },
        card: {
          DEFAULT: 'hsl(var(--card))',
          foreground: 'hsl(var(--card-foreground))',
        },
      },
    },
  },
};
```

## 4. 사용 예시

### React + Tailwind 예시

```tsx
// 헤더 컴포넌트
<header className="bg-card border-b border-border">
  <span className="text-foreground">WEAV AI</span>
  <button className="hover:bg-accent text-primary-foreground">
    메뉴
  </button>
</header>

// 카드 컴포넌트
<div className="bg-card text-card-foreground border border-border rounded-lg p-4">
  <h2 className="text-foreground">제목</h2>
  <p className="text-muted-foreground">설명 텍스트</p>
</div>

// 버튼 컴포넌트
<button className="bg-primary text-primary-foreground hover:bg-accent">
  클릭
</button>
```

## 5. 디자인 특징

- **다크 테마**: 매우 어두운 배경 (`hsl(222.2, 84%, 4.9%)`)과 밝은 텍스트 (`hsl(210, 40%, 98%)`)의 대비
- **차분한 색감**: 채도가 낮은 회색-청색 계열의 보조 색상 사용
- **명확한 계층**: Background → Card → Accent 순서의 명확한 색상 계층
- **접근성**: 충분한 대비율로 가독성 확보
